s ↔ h 対応（ラテン語 vs ギリシャ語）

印欧祖語の語頭 *s がラテン語ではそのまま s として残り、ギリシャ語では気息音 h に変化した。同じ概念を表す形態素がラテン語とギリシャ語で異なる音を持つため、英語には両方が接頭辞として残っている。

グリムの法則により、英語の日常語（father・fish・foot・heart など）とラテン語・ギリシャ語由来の学術語（paternal・piscine・pedal・cardiac など）が同語源のペアを形成している。英語の語彙の二重構造の主要な原因のひとつ。

ヴェルネルの法則

1875年にカール・ヴェルネルが発見した、グリムの法則の例外を説明する規則。グリムの法則通りに無声音になるはずの子音が、印欧祖語のアクセントの位置によって有声音になる場合がある。

規則の概要
グリムの法則では *p→f、*t→þ、*k→h になるはずだが、印欧祖語でその子音の直前の音節にアクセントがなかった場合、無声摩擦音ではなく有声音（f→v、þ→d、h→g/w）になった。つまりアクセントの位置が子音の変化先を決めていた。
アクセントなし：*t → þ（th）→ さらに d に有声化
対応単語説明
f → v have / had
life / live
wolf / wolves
knife / knives		 語中・複数形など有声音に挟まれる位置で -f- が -v- に有声化。have の -v- と had の -d- は同じ語根から生まれた二形。life/live・wolf/wolves は単複や派生で f/v が交替する。
þ → d（th → d） father（pater）
brother（frater）		 ラテン語 pater の -t- はグリムの法則では -þ-（th）になるはずだが、英語では father と -ð-（d音）になっている。印欧祖語でアクセントが第2音節（-tḗr）にあったためヴェルネルの法則が適用された。一方 brother（frāter）はアクセントが第1音節にあったため -þ-（th）が保持された。
s → z → r
（ロータキズム）		 was / were
lose / forlorn		 ヴェルネルの法則で -s- が -z- に有声化し、さらに西ゲルマン語で -z- が -r- に変化（ロータキズム）。was はアクセント前の -s- が保持、were の -r- はアクセント後の -s- が z→r と変化した形。forlorn の -r- も同様。
同語根の分化 birth / burden
seethe / sodden		 同じ語根から、アクセント位置の違いで子音が異なる二形が生まれた。birth と burden は同語源だが -th- と -d- が交替。seethe（煮立つ）の過去分詞 sodden（びしょ濡れ）も -th-/-dd- の交替による。

ヴェルネルの法則は比較言語学の歴史上重要な発見で、「例外のない音変化の法則は存在する」という青年文法学派の主張を支持する証拠となった。グリムの法則の「例外」に見えた現象が、実はより深い規則性を持っていたことを示した。

英語の三層構造

英語の語彙は大きく三つの層に分かれる。古英語・ゲルマン語由来の日常語、1066年のノルマン征服後に流入したフランス語由来の中間層、そしてラテン・ギリシャ語由来の学術語。三層はそのまま格式の高さに対応しており、同じ概念でもニュアンスが異なる。

なぜフランス語層が生まれたか
1066年のノルマン征服により、フランス語を話すノルマン人が支配層となった。以降約300年、法律・宮廷・教会の公用語はフランス語で、英語は庶民の言語にとどまった。この結果、支配・統治・文化に関わる語彙がフランス語から大量に流入し、ゲルマン系の日常語と並存することになった。

格式による三層対応

左から右へ、日常的・親しみやすい → 礼儀正しい・フォーマル → 学術的・堅い、というニュアンスの変化がある。

料理の語彙：育てる人と食べる人

牛・豚・羊を育てるのはゲルマン系の農民で、食べるのはフランス語を話す貴族だった。生きた動物の名前はゲルマン系、調理後の肉の名前はフランス語由来という分裂が生じた。
